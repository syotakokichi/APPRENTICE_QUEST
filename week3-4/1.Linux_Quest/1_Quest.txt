①Linux 
Linux の基本コマンドが使え、簡単なシェルスクリプトが書けるようになることを目指します。

②Git/GitHub
Git と GitHub の基本操作を身につける。最終的には GitHub フローに従って開発を進められるようになることを目指す。

Week1-2でできなかったadvanceの知識を身につける。演習問題などを実施して文章からコードにするまでの流れをスムーズにできるようにする。

AWSに関する知識が皆無なので、Udemyの講座(ゼロから実践するAWS)を終わらせ、基本的な知識を身につける。
