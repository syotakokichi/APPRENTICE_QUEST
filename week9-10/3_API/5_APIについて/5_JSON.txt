JSON は API のリクエストとレスポンスで一般的に用いられるデータ形式です。以下のデータを JSON の形式で表現してください。

名前: 山田太郎
年齢: 20

指定されたデータをJSON形式で表現すると次のようになります：

json
Copy code
{
  "名前": "山田太郎",
  "年齢": 20
}
JSON（JavaScript Object Notation）は、データを軽量で人間が読みやすく、マシンが解析や生成しやすい形式で表現するための標準です。主に二つの構造を基にしています：

名前/値のペアの集まり：これは様々な言語で "オブジェクト"、"レコード"、"構造体"、"辞書"、"ハッシュテーブル"、"キー付きリスト"、"連想配列" などと呼ばれます。
値の順序付きリスト：これはほとんどの言語で "配列"、"ベクトル"、"リスト"、"シーケンス" などと呼ばれます。
上記の例では、"名前"と"年齢"がキーで、"山田太郎"と20がそれぞれのキーに対応する値です。これらのキーと値のペアがオブジェクトを形成しています。
